# Chart Selection Matrix

Use this matrix to choose chart form quickly.

| Question Type | Primary Task | Recommended Chart | Fallback |
|---|---|---|---|
| Which category is bigger/smaller? | Compare magnitudes | Horizontal bar | Dot plot |
| How did this metric change over time? | Trend | Line | Area (if cumulative emphasis needed) |
| How do two groups compare before/after? | Side-by-side comparison | Paired bars | Slope graph |
| What drives the total composition? | Part-to-whole | 100% stacked bar | Treemap |
| Are two variables related? | Correlation/outliers | Scatter | Bubble (third variable) |
| What exact values are required? | Lookup precision | Table | Labeled bars |

## Guardrails

- Bars/columns should start at zero.
- Avoid pies except in strict low-slice scenarios.
- Prefer direct labeling and horizontal category text.
