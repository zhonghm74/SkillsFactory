# Anomaly RCA Playbook

## 7-Step Diagnostic Sequence

1. Reproduction
   - Reproduce impacted path or business flow.
2. Data Verification
   - Verify event integrity and reporting lag across systems.
3. Helicopter View
   - Compare against historical baseline and seasonality.
4. Context Synthesis
   - Review releases, campaigns, incidents, and external events.
5. Slicing & Dicing
   - Segment by region, cohort, channel, product, platform.
6. Metric Decomposition
   - Break ratio metrics into numerator/denominator and funnel stages.
7. Synthesis & Conclusion
   - Test if removing suspect variable explains shift.

## Guardrail
Do not propose root cause until all seven steps are addressed.
