# Indicator / Driver Tree Playbook

## Purpose
Translate strategic goals into operational levers teams can control.

## Structure
- Strategic Objective -> Theme -> Tactical Driver -> Metric
- Each metric must have:
  - owner
  - threshold corridor
  - action trigger condition

## Design Rules
- Keep branches MECE where possible.
- Explicitly mark conflicts and trade-offs (speed vs quality, growth vs retention).
- Separate leading and lagging indicators.
- Use dotted relation markers for unverified hypotheses.

## Output Standard
A valid driver tree should allow a reviewer to trace any bottom metric to a single top objective.
