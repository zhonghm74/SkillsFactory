# Minto Pyramid Reporting

## Core Structure
- Top: Assertion (BLUF)
- Middle: Supporting arguments (MECE)
- Bottom: Evidence (data, diagnostics, tests)

## Narrative Arc
Situation -> Complication -> Question -> Answer

## Writing Rules
- Lead with recommendation, not chronology.
- Group arguments by decision logic, not by data source.
- Keep each argument backed by explicit evidence.
- End with action plan (owner, timeline, expected impact).

## Minimum Report Sections
1. Executive Assertion
2. Key Supporting Arguments
3. Evidence Summary
4. Risks and Confidence
5. Recommended Actions
